# Customer Churn Prediction

This project predicts whether a customer will churn using machine learning
classification techniques.

## Problem Statement
Customer churn is a major challenge for service-based companies. This project
uses historical customer data to predict churn and identify key factors
affecting customer retention.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Model Used
- Logistic Regression with Scikit-learn Pipeline (StandardScaler + Model)

## Key Features
- Data cleaning and preprocessing
- Categorical encoding
- Feature scaling using Pipeline
- Model evaluation (Accuracy, Confusion Matrix)
- Identification of key churn factors

## Result
The model achieves good accuracy and highlights important churn drivers such as
contract type, tenure, and monthly charges.

## Conclusion
This model helps businesses identify customers at risk of churn and take
proactive retention actions.
